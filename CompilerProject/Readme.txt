===============================

Included in this release:

   * Lexical analysis
    
To build, at the terminal type:

    make
   
To run, type:

    ./LegendaryTokenAnalyzer.exe <file_name>
    
Where <file_name> is the name of a LegendaryTokenAnalyzer source file. You can try with
these files:

   * binary.deep
